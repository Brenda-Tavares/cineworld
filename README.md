## Features

- Advanced movie search with AI-powered recommendations
- Genre filtering (20+ categories)
- Custom sorting (top rated, latest, upcoming, lowest rated)
- Country and year filters
- Multi-language support (Portuguese, English, Spanish, Chinese, Japanese, Russian, Korean)
- Streaming provider information
- Responsive design (mobile, tablet, desktop)
- Integrated contact system
- Advanced security (rate limiting, CORS, security headers)
- Automatic deployment on Vercel

## Live Demo

Access: [https://cineworld-site.vercel.app](https://cineworld-site.vercel.app)

## Technologies

- **Front-end:** HTML5, CSS3, JavaScript (Vanilla SPA)
- **Back-end:** Node.js, Express.js (local), Vercel Serverless Functions (production)
- **API:** TMDB (The Movie Database), Google Gemini AI
- **Database:** Supabase (PostgreSQL)
- **Hosting:** Vercel
- **Security:** Helmet, Rate Limiting, CORS, Input Validation
- **External:** Font Awesome, Google Fonts, Google AdSense

## Environment Variables

Create a `.env` file in the root directory:

```
TMDB_API_KEY=your_tmdb_api_key
GEMINI_API_KEY=your_gemini_api_key
```

## Local Development

```bash
# Install dependencies
npm install

# Start Vercel development server
vercel dev
```

The app will be available at `http://localhost:3000`.
